> # Institute of Digital Risk (IDR) – Homepage Design

This project was developed as part of a **Web Development Internship assignment for AKOIN**.

The goal of this assignment was to design a **simple brand identity and responsive homepage** for the **Institute of Digital Risk (IDR)** — an industry-led institute focused on training and deploying professionals in **digital risk, cybersecurity, and AI governance**.

The project demonstrates **UI design thinking, responsive frontend development, and clean semantic web architecture** using vanilla web technologies.

---
> ## Live Demo
> View the deployed website: https://ankitdimri4.github.io/idr-homepage-design/

---

# Project Overview

The website introduces the **Institute of Digital Risk** and clearly communicates:

* The institute’s mission
* Its training and innovation model
* The community it serves
* A simple contact section for inquiries

The design follows a **minimal technology-focused visual style** using the required color palette:
* Orange
* White
* Black

The layout is designed to be **clean, modern, and readable across desktop and mobile devices**.

---

# Features

* Responsive layout for **desktop and mobile**
* Sticky navigation bar
* Smooth scrolling navigation
* Modern UI with CSS hover effects and animations
* Structured sections explaining the IDR model
* Contact form layout
* Clear visual pipeline for the **Train → Hire → Innovate → Deploy model**

# Website Sections

### Hero Section

Introduces the institute’s mission with a clear headline and call-to-action.

### About IDR

Explains the role of the institute and how it connects academic insight with real-world digital risk practice.

### Service Pillars

Highlights the three main components of the IDR model:

* **Academy**
* **Innovation & Incubation**
* **Advisory Services**

### Model Pipeline

A visual representation of the process:

```
Train → Hire → Innovate → Deploy
```

### Community

Describes the learners and organisations that participate in the IDR ecosystem.

### Contact

A responsive contact section with a simple inquiry form.

# Logo Design

The logo was designed using **Figma** and follows the assignment requirement of a **cube-inspired geometric symbol**.

Two variants were created:

* Icon-only version
* Icon + text version (Institute of Digital Risk)

The design remains **legible across different sizes**, including favicon and mobile header usage.

### File Description

Contains the semantic structure of the webpage including all sections.

**style.css**
Handles layout design, responsive styling, animations, and visual components.

**script.js**
Implements interactive features such as smooth scrolling and mobile navigation behavior.

**assets/**
Contains branding assets and illustrations used in the interface.

**logo-explanation.txt**
Contains a short explanation describing the logo design choices.

# Responsive Design

The website uses:

* **CSS Flexbox**
* **Media queries**

This ensures the layout adapts properly across:

* Desktop screens
* Tablets
* Mobile devices

Desktop sections use **multi-column layouts**, while mobile screens stack content vertically for readability.
